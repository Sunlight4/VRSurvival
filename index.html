Download(zip file)
Disk Image for Mac 10.5 or later
Download the VSurvival zip archive or dmg. Unzip it and you will have a folder that is named Download

1. Open the Download folder
2. If you don't have Python 2.7 and Pygame, double click the files that end with .mpkg
3. Use the .mpkg files to install Python and Pygame.
4. Open the VSurvival (1) Folder.
5. Inside is a VSurvival folder. Drag that to your home folder.
6. Put VSurvival.command to your home folder too.
7. Open a Command Line window.
8. Run "chmod a+x -R ~/VSurvival"

To run VSurvival double-click the VSurvival.command file.

Troubleshooting:

If you get an error that says: ImportError no module named pygame, YOU DIDN'T INSTALL PYGAME CORRECTLY!!!!
